1️⃣ What is the difference between var, let, and const?

ans:-

1️// var

a/Function scoped (function এর ভিতরে সীমাবদ্ধ)

b/একই variable পুনরায় declare করা যায়

c/value change করা যায়

2️// let

a/ Block scoped ( { } block এর ভিতরে সীমাবদ্ধ )

b/ একই variable পুনরায় declare করা যায় না

c/ value change করা যায়

3️// const

a/ Block scoped

b/ পুনরায় declare করা যায় না

c/ value change করা যায় না

3️⃣ What is the difference between map(), filter(), and forEach()

ans:-

JavaScript-এ map(), filter(), এবং forEach() — এই তিনটি array method

1️// map

a// Array এর প্রতিটি element এর উপর কাজ করে

b// নতুন array return করে

c// সাধারণত data modify / transform করতে ব্যবহার হয়

2️// filter()

 a// Condition অনুযায়ী element select করে

b/  নতুন array return করে

c//শুধু সেই element গুলো রাখে যেগুলো condition true হয়

3️// forEach()

a// Array এর প্রতিটি element এর উপর loop চালায়

b// কোনো নতুন array return করে না

c// সাধারণত side work (console log, DOM change) এর জন্য ব্যবহার হয়
